# Your Energy

Навчальний проєкт з дисципліни «Основи веброзробки (JavaScript)». Автор: Soroka Volodymyr.

## Запуск локально

1. `npm install`
2. `npm run dev`

## Збірка

- `npm run build` створює продакшн-версію у `docs/` для GitHub Pages.

## Деплой

- Гілка: `gh-pages`
- URL: https://vsoroka-hash.github.io/js-course-final/

## Оптимізація зображень

- `npm run optimize:images`
